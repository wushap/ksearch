## 发现的问题

### 1. 某些关键词无结果
- "linux systemd service" 返回0条结果
- 可能原因：SearXNG 无匹配结果或URL被跳过

### 2. 内容质量
- 部分结果内容较短（被清理过滤）
- markitdown 对某些网站转换失败

### 3. 慢网站
- 被跳过的URL：YouTube, sputniknews.cn
- 转换超时的URL：guancha.cn

## 推荐配置

基于测试结果，推荐以下配置：

```json
{
  "max_results": 5,
  "timeout": 30,
  "format": "markdown"
}
```

**理由**:
- max_results=5: 平均10秒完成，结果充足
- timeout=30: 足够处理大多数网站
- format=markdown: 包含完整内容便于阅读
